 🔐 Lock Code Challenge

A premium cyber security themed puzzle game built using HTML, CSS, and Vanilla JavaScript.

Test your logical thinking, speed, and decision making skills by cracking dynamically generated lock codes before time runs out. Featuring a futuristic cyberpunk inspired interface, mission based progression, achievements, statistics tracking, and responsive gameplay across desktop, tablet, and mobile devices.


 🚀 Features

 🎮 Core Gameplay

* Multiple difficulty levels
* Dynamic lock code generation

* Mission based progression

 🎯 Gameplay Rules

1. Select a difficulty level or mission.
2. Start the challenge.
3. Enter the correct lock code.
4. Attempts and time are limited.
5. Only one hint can be used per game.
6. After each attempt, the security system generates a new secret code.
7. Earn XP and achievements by completing missions.
8. Improve your rank by achieving higher scores.



 🛠️ Technology Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
* LocalStorage API

No frameworks.
No backend.
No external dependencies.



 ▶️ Getting Started

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Select a difficulty level.
4. Start the game.
5. Crack the lock code and complete missions.

No installation required.
